## 安全特性

### 1. 认证授权
- 所有接口都需要 JWT token 认证
- 用户只能修改自己的信息
- 中间件自动验证 token 有效性

### 2. 密码安全
- 密码强度验证（至少 8 位，包含字母和数字）
- 修改密码时必须验证旧密码
- 使用 Argon2 算法哈希存储
- 登录防暴力破解（5 次失败锁定 5 分钟）

### 3. 文件上传安全
- 前端验证文件类型（仅图片）
- 前端限制文件大小（2MB）
- 后端生成唯一文件名（防止覆盖）
- 文件存储在独立目录

### 4. 输入验证
- 前端表单验证
- 后端参数验证
- SQL 注入防护（ORM 自动处理）
- XSS 防护（前端框架自动转义）
